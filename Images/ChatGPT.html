how to find the index of a row in a dataframe using pandas for a row where 2 columns match some data?
ChatGPT

You can find the index of a row in a pandas DataFrame where two columns match specific data by using the loc method along with boolean indexing. Here's an example:

python
import pandas as pd

# Create a sample DataFrame
data = {'Column1': [1, 2, 3, 4],
        'Column2': ['A', 'B', 'C', 'D'],
        'Column3': ['X', 'Y', 'Z', 'W']}
df = pd.DataFrame(data)

# Specify the values you want to match
value1 = 2
value2 = 'B'

# Find the index where both conditions are met
index = df.loc[(df['Column1'] == value1) & (df['Column2'] == value2)].index

# Check if the index is not empty before accessing the value
if not index.empty:
    result_index = index[0]
    print(f"The index where Column1 equals {value1} and Column2 equals '{value2}' is: {result_index}")
else:
    print("No matching row found.")

In this example, the loc method is used with boolean indexing to filter rows where both conditions are true ((df['Column1'] == value1) & (df['Column2'] == value2)). The resulting DataFrame contains only the rows that satisfy both conditions, and then you can access the index of the first (and only) matching row.

Make sure to adjust the column names, values, and conditions according to your specific DataFrame structure and data.
